Toxicjasmine: A simple ci runner for jasmine tests
==================================================

To install toxicjasmine use:

.. code-block:: sh

   pip install toxicjasmine --extra-index-url=https://pypi.poraodojuca.dev


To run your tests:


.. code-block::

   toxicjasmine ci path/to/you/SpecRunner.html
